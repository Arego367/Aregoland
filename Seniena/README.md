# Aregoland — Arego Chat

Aregoland entstand aus einer einfachen Frage: **Warum gibt es keine App, der man wirklich vertrauen kann?**

Als alleinerziehender Vater von zwei Toechtern im Grundschulalter wollte ich einen Ort schaffen, an dem Familien sicher miteinander kommunizieren koennen — ohne Datenhunger, ohne Algorithmen die suechtig machen, ohne Kompromisse beim Datenschutz. Aregoland ist kein Startup. Es ist ein Projekt aus Ueberzeugung, gebaut in den Abendstunden neben einem Vollzeitjob.

## Was ist Arego Chat?

- P2P Messenger — Nachrichten gehen direkt von Geraet zu Geraet
- E2E verschluesselt — kein Server speichert Inhalte
- Spaces — Raeume fuer Familie, Vereine, Firmen mit Rollen & Rechten
- Kalender, World, Pay — alles in einer App
- Passwordless — kein Passwort, dein Geraet ist dein Schluessel
- Datenschutz by Design — keine Cookies, kein Tracking, keine Werbung
