# EUDI Wallet Integration — Aregoland

> Stand: 2026-04-01

## Was ist EUDI fuer Aregoland

EUDI Wallet ist das technische Fundament von Aregoland.
Kein eigenes Zahlungssystem. Kein BaFin. Keine Abhaengigkeit.
Die EU liefert die Infrastruktur — Aregoland liefert den Kanal.

## Was EUDI fuer Aregoland ermoeglicht

- Identitaet: Nutzer verifiziert sich per EUDI — sicher, EU-weit
- FSK-Verifizierung: Geburtsdatum kryptografisch bestaetigt
- Dokumente: Befunde, Zeugnisse, Antraege P2P ueber EUDI
- Zahlung: EUDI Wallet Zahlfunktion ab 2027
- Alles in einer Infrastruktur — kostenlos fuer Nutzer

## FSK-Automatik

Am Geburtstag schaltet EUDI das Alter frei:
- 12. Geburtstag → FSK 12 automatisch aktiv
- 16. Geburtstag → FSK 16 automatisch aktiv
- 18. Geburtstag → FSK 18 automatisch aktiv

Kein Elternteil noetig. Kein Formular. Kein Antrag.
EUDI weiss das Geburtsdatum — Aregoland fragt es ab.

## Warum EUDI

- Kostenlos fuer Nutzer und Anbieter
- EU-Standard — keine Abhaengigkeit von privatem Anbieter
- DSGVO by Design — in der Verordnung verankert
- Open Source — jeder kann pruefen
- Passt perfekt zur Aregoland-Philosophie
